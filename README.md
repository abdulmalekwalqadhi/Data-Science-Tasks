# Data Science Internship Tasks

## Overview
This repository contains three completed data science tasks involving data analysis, visualization, and machine learning.

---

## Task 1: Iris Dataset Visualization
### Objective
Explore and visualize dataset features.

### Approach
- Loaded dataset using pandas
- Created scatter plots, histograms, and boxplots

### Results
- Clear separation between species
- Petal length is the most important feature

---

## Task 2: Credit Risk Prediction
### Objective
Predict loan approval status.

### Approach
- Data cleaning and preprocessing
- Logistic Regression model
- Feature encoding and scaling

### Results
- Accuracy ≈ 75%
- Confusion matrix shows strong prediction for approved loans

---

## Task 4: Insurance Cost Prediction
### Objective
Predict insurance charges.

### Approach
- Linear Regression model
- Feature analysis using visualization

### Results
- Smoking and age strongly influence cost
- Model performs reasonably well

---

## Conclusion
These tasks demonstrate skills in:
- Data preprocessing

- Machine learning modeling
